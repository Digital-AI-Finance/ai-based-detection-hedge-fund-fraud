# Contributing

Thank you for your interest in contributing to this repository.

## How to Contribute

### Slides
- All slides use the shared `template_beamer_final.tex` Beamer template
- Follow the Madrid theme conventions (8pt, 16:9 aspect ratio)
- Use notation macros from `notation.tex`

### Charts
- Python scripts must be standalone and self-contained
- Use matplotlib + seaborn for visualization
- Follow the color scheme: mlblue, mlpurple, mllavender
- Output both PDF and thumb.png (150 DPI)

### Quizzes
- Each section has a `questions.json` file
- Questions must be factually traceable to the paper source
- Include section references for all answers

## Code Style
- Python: Follow PEP 8
- LaTeX: Use consistent formatting and commenting

## Reporting Issues
Please open a GitHub issue for bugs, corrections, or suggestions.
